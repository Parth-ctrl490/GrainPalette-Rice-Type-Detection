About the Rice Type Classification Model

This model is built using Convolutional Neural Network and trained on labelled images of rice. The model takes image of rice grain as input and gives its prediction for the type of rice.

Accuracy of the Model

The model is capable of checking for 5 different rice types.

Different Types of Rice

Arborio, Basmati, Ipsala, Jasmine, Karacadag

Dataset used

The dataset of labelled images is obtained from Kaggle.

Technical Architecture

The model is trained on MobileNetV2 Architecture using Transfer Learning.
